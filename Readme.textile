h1. ブログアプリのサンプル

h3. 必要なもの

Sqlite3

PHP5.2 >

PHP PDO Sqlite


h3. インストールの例


ドキュメントルート以下で、
<pre>
	$git clone git://github.com/swimmystudy/jenkins-flow.git
</pre>
http://localhost/jenkins-flow/

にアクセスすれば確認できると思います。


h3. テストの実行

ダウンロードしたフォルダに移動
<pre>
	$cd jenkins-flow
	$lib/Cake/Console/cake testsuite app AllTests --stderr
</pre>
で確認できると思います。
